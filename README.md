# CreditWise - Loan Approval Analyzer

Live Demo: [Click Here](https://adeebanizam22-creditwise--loan-approval-analyzer-app-0whqle.streamlit.app/)

An end-to-end machine learning project that predicts loan approval using multiple classification algorithms.

## What this project does
- Performs EDA and handles missing values
- Trains and compares 3 models: Logistic Regression, KNN, Naive Bayes
- Selects Naive Bayes as final model based on highest precision (81.1%)
- Deployed as an interactive web app using Streamlit

## Tech Stack
Python · Pandas · Scikit-learn · Naive Bayes · Streamlit
